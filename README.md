# KASH Shell
Implementation of a basic shell in C.

This will not work on Windows under the MinGW environment preferably try the Cygwin or WSL.
This is a simple cmake project.

## Build Instructions
```
$ mkdir build
$ cmake -S . -B build/
$ cd build
$ make
```
```
$ ./kash
```
